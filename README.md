# Угадай число: Игра и алгоритмы

Это учебный проект для освоения Python, стиля кода PEP8 и работы с системой контроля версий Git/GitHub.

Проект содержит две реализации игры "Угадай число":
1. **game.py** – Игра, где компьютер угадывает число методом случайного подбора.
2. **game_v2.py** – Усовершенствованная версия, где компьютер использует алгоритм бинарного поиска, чтобы угадать число не более чем за 7 попыток.

## Как это работает

*   **game.py:** Компьютер последовательно генерирует случайные числа в заданном диапазоне (1-100) до тех пор, пока не найдет загаданное.
*   **game_v2.py:** Компьютер использует стратегию бинарного поиска. Он постоянно делит диапазон возможных чисел пополам и выбирает среднее значение, что позволяет найти ответ за минимальное количество шагов (log₂N).
